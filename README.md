 StateWise Business Analytics Dashboard

An interactive **Business Performance & Revenue Analytics Dashboard** built using Power BI. This dashboard provides deep insights into revenue, units sold, marketing spend, and customer distribution across different states.

---

##  Features

-  State-wise performance analysis  
-  Revenue, units sold, marketing spend & customers tracking  
-  Interactive filtering (click on any state to update all visuals)  
-  Data-driven insights for better decision-making  
-  Clean and user-friendly dashboard design  

---

##  Dashboard Overview

The dashboard includes:

- **KPIs**
  - Total Revenue  
  - Units Sold  
  - Marketing Spend  
  - Unique Customers  

- **Visualizations**
  - Revenue by State (Bar Chart)  
  - Units Sold Trend by State (Line Chart)  
  - Marketing Spend vs Revenue (Scatter Plot)  
  - Customer Distribution (Donut Chart)  
  - Geographic Map View
